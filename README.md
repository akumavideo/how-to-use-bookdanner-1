## ブック・ダナーのつかいかた

1. LINEのメッセージ入力欄左にある`+`を押して`カメラ`を選択！

1. カメラが起動するので、本の背表紙を撮影してそのまま画像を送ってね  
    ヒント：5冊くらいがいちばん認識率が高いよ
1. 後は待ってればブック・ダナー君が本を探してきてくれます！

※画像に写ってない本が出ることもあるけど、できるだけ同じジャンルの本を表示できるようがんばります！！
